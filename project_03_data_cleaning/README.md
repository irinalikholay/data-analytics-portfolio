# Project 03 - Data Cleaning with Python 

## Business goal
Clean and prepare customer data for analysis.

## Dataset
Raw customer dataset with missing values and inconsistencies.

## Tools
- Python 
- pandas

## Status
In progress

## Data quality issues identified 

- Missing values in age, gender, country, signup_date and spend
- Invalid age values ( e.g. very high age)
- Negative spend values
- Dates stored as text instead of date format
