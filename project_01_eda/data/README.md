
# Progect 01 - Exploratory Data Analysis (EDA)

## Project description 
This project is an introductory exploratory data analysis using Python and pandas.
The goal is to understand the dataset structure and calculate basic metrics.

## Dataset
The dataset contains information about customers:
- customer_id - unique customer identifier 
- age - customer age 
- country - customer country 
- purchase_amount - purchase amount

## Analysis steps 
The following steps were performed:
- Dataset structure inspectiom
- Calculation of average customer age 
- Identification of minimum and maximum age 
- Calculation of average purchase amount by country 

## Key insights 
- The average customer age was calculated 
- Customer ages were validated using minimum and maximum values
- The average purchase amount differs between countries
